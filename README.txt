# Permanent Password
A light-weight plugin to add a permanent password to any user on your website.

## Installation
+ Download the file **permanent-password.zip**
+ Go to **Dashboard > Plugins > Add New**
+ Click on the **Upload Plugin** button
+ Upload the zip file you downloaded earlier
+ Click on **Install Now** button and then **Activate** the plugin
